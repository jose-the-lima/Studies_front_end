Mobile first e alguns tipos de layouts

Mobile first

É um conceito que trata de criar um site pensando "primeiro no dispositivo móvel", ou seja trata-se de fazermos um site mas pensando primeiro em como ele vai se comportar no smartphone.

Então quer dizer que eu tenho que desenvolver o site primeiro em uma tela pequenina para depois ir ver como ele funciona no computador ? Não. Esse conceito tem muitas interpretações e cada um vai entender de uma forma, mas basicamente, você vai fazer o seu site primeiro para o computador, mas a todo momento pensando em como ele vai se comportar na menor tela possível, no caso um smartphone ou afins.

Sempre pensando em como ele vai se comportar na Maior tela e também na Menor tela disponível.

No mundo do desenvolvimento front-end temos alguns tipos de desenvolvimento e layouts, vai daquele que em qualquer situação que não seja a dele vai quebrar tudo até aqueles que se adaptam a qualquer tipo de desafio. Vejamos alguns desses tipos abaixo:

Tipos de layouts

  • Normal
  • Responsivo
  • Adaptativo

Normal

O normal como já se diz, vai apenas seguir o fluxo de desenvolvimento para o qual ele foi feito e não vai se adaptar a nenhum outro.

Responsivo

Esse tipo vai ser responsivo a qualquer tipo de tela que abrir ele, ou seja ele vai responder a qualquer tela que o abrir de uma forma ótima e vai ser sempre assim e nunca vai quebrar.

Adaptativo

O layout adaptativo vai se adaptar a determinados tipos de tela, ou seja ele vai ter um "tamanho" exclusivo para aquela dimensão atual, mas quando diminuir ao certo ponto de não dar mais, ele vai trocar para o estilo daquela dimensão, ou seja vai mudar os tamanhos para aquela situação ficando assim adaptado.

Qual é o melhor no fim das contas ? geralmente o melhor a se fazer é sempre ir para um "responsivo/adaptativo", ou seja unir os dois em um só, fazer um layout adaptativo que seja responsivo até um determindo ponto e quando não der mais, ele se adapta novamente e novamente... Geralmente os tipos de tamanhos mais utilizados para definir a adaptação são:

  • 320px
  • 480px
  • 760px
  • 960px
  • 1200px
  • 1600px
