Formatando Textos

A formatação de textos pode ser muito feita com o próprio CSS, mas o problema nisso, é que o "CSS" não tem o poder de deixar partes do texto com a devida "importância" que ele precisa para ser apresentado aos motores de busca.
O CSS vai cuidar da estilização em si, mas apenas isso, a parte de importância e destaque aos motores de busca e leitores de tela é o próprio HTML quem vai cuidar, então temos algumas tags importantes que fazem essa formatação e ainda dão essa importância necessaria:

  • <strong> : Utilizada para deixar o texto em negrito e com mais "importância" para leitores de tela e motores de busca.
  • <em> : Tag que vai deixar o conteúdo em itálico, feita para dar uma "enfase" a mais para o texto.
  • <small> : Tag utilizada para dar "menos importância" ao conteúdo que está dentro dela, é basicamente um reverso do strong.
  • <mark> : Tag utilizada como um "marca-texto" vai marcar o conteúdo com um fundo colorido e vai já deixar com uma importância tanto visual quanto para leitores de tela
  • <del> : Utilizada para "deletar" um elemento do texto, não vai fazer a remoção em si, mas vai dizer tipo "ignore essa parte"
