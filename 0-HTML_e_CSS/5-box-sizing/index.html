Box-sizing

Basicamente o box-sizing, vai definir o tamanho que a caixa vai seguir, se vai ser o tamanho real definido por você, ou se vai ser um tamanho relativo a todos as própriedades adicionadas, temos 2 valores importantes:

  • content-box

    Vai somar o tamanho do conteúdo que você definiu + borda + padding.

  • border-box

    Vai subtrair do tamanho que você definiu o valor da: borda e padding. Para que o tamanho final continue sendo o valor definido desde o inicio.
