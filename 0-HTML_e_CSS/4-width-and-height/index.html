Width e height

Podemos definir largura e altura de várias formas diferente, mas é claro sempre utilizando primeiramente as seguintes própriedades CSS:

  • width
  • height
  • max-width - min-width
  • max-height - min-height

Podemos definir o seu valor em pixels, porcentagem, "auto"(Que é o tamanho automático do elemento), "inherit"(Herda o mesmo valor do elemento pai)

Definindo tamanhos com medidas fixas(Pixels)

Por padrão não irão ser nada responsivas quando a tela diminuir

Definindo tamanhos com medidas relativas(%)

Vão se adaptar aos elementos aos quais estão armazenadas

As própriedades "min e max" vão definir um tamanho mínimo para o elemento e um tamanho máximo que ele pode alcançar respectivamente, eles sempre terão "prioridade" frente ao width ou height comum.
