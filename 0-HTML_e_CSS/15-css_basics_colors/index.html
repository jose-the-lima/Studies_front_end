Colors

Temos várias formas de definir cores no CSS, seja pelo nome da cor ou com outras funcionalidades, como funções, hexadecimal...
Os principais modos de definir cores e os que são mais utilizados são:

  • Nome da cor(Em inglês)
  • RGB
  • hexadecimal
  • palavra-chave(currentColor)
