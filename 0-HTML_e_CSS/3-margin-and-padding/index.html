Margin e padding

Margin é o espaçamento externo de um elemento, ou seja fora do elemento é ele quem vai empurrar eles de um canto para o outro sem afetar no tamanho.

Sem margin:

Padding é o espaçamento inteiro do elemento, é ele quem vai ser o espaço extra entre o conteúdo e a borda que vai definir um respiro para o elemento
